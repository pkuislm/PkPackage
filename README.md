# PkPackage
## 简易的补丁文件管理

功能：将补丁文件打包至单一文件中，且对补丁内的文件的版本进行控制
（不过实际上也只是个小玩具罢了，局限性很大）

# 注意：补丁文件不含加密
详情参考代码及代码内注释


目前使用了本套代码且已发布的游戏：[绿茶汉化组]天津罪


PkPackageLib：请编译为静态库（lib），在补丁代码中#include lib.h即可  
PkPackageMaker：用来制作封包的工具（内含解包逻辑），请编译为控制台应用
